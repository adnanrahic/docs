[Sematext](https://sematext.com/) is a full-stack observability tool that helps developers keep track of logs, metrics, infrastructure, and website performance.

This guide walks you through setting up client-side user experience monitoring for your <OrgName /> deployments. Once you add the script in the `<head>` tag of your website/front-end app, it will automatically send metrics to Sematext.

You'll be able to debug and troubleshoot performance issues and end-user satisfaction with Web Vitals, Apdex scores, Waterfall charts showing slow-loading resources, pages, HTTP requests, and much more.
